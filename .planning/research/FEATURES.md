# Rich Text Editor Features

**Analysis Date:** 2026-03-02

## Table Stakes (Must Have)
- **Formatting**: Bold, Italic, Underline (Standard document controls).
- **Paragraphs**: Alignment (Left/Center/Right) and Headers (H1-H3).
- **Lists**: Bulleted and Numbered lists.
- **Links**: Basic hyperlink creation and editing.
- **Toolbar**: Classic top-mounted sticky toolbar.

## Differentiators (Value Add)
- **Tables**: Row/column manipulation and styling.
- **Media**: Drag-and-drop image uploads + captioning.
- **Undo/Redo**: Deep history tracking.
- **Markdown Support**: Auto-completion and Markdown export.
- **Collaboration**: Real-time sync (Crdt/Websocket).
