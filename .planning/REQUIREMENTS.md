# Project Requirements

**Vision:** A premium, modular Word-style rich text editor with zero dependencies.

## Key Goals
1. **Classic Word Interface**: Sticky toolbar with core formatting.
2. **Modular Philosophy**: Every feature is an independently loadable plugin.
3. **Clean Output**: Generate clean, semantic HTML on every save.

---

## Active Requirements (Milestone 1)

### 1. Typography Core (P0)
- **REQ-01**: Implement Bold, Italic, Underline, and Strikethrough formatting.
- **REQ-02**: Support Font Family and Font Size selection from the toolbar.
- **REQ-03**: Support Text Color and Background Highlight selection.

### 2. Paragraph Management (P0)
- **REQ-04**: Core alignment controls (Left, Center, Right, Justify).
- **REQ-05**: Header support (H1, H2, H3, P).

### 3. Lists & Structure (P1)
- **REQ-06**: Bulleted and Numbered lists with basic nesting.
- **REQ-07**: Horizontal Line (HR) insertion.

### 4. Interactive Elements (P1)
- **REQ-08**: Hyperlink creation and management via a modal dialog.
- **REQ-09**: Basic image insertion through the plugin layer.

### 5. Utilities & DX (P2)
- **REQ-10**: Core keyboard shortcuts (Ctrl+B, Ctrl+I, etc.).
- **REQ-11**: Clear formatting function to revert selected text.

---

## Out of Scope (Milestone 1)
- Advanced Table merging (just basic creation).
- Drag-and-drop file uploads.
- Real-time collaboration.
